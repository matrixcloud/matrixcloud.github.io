古人云：“读万卷书，行万里路。”。乘换工作之机，一者为增长见闻，二者为放松身心。约友，无一人，遂于蚂蜂窝订花儿户外拼车票一张，只身从上海出发。至杭州一宿，与友借单反把玩。及西宁又一宿，翌日清晨与师傅及三位伙伴会合，至此，旅途正式开始。

{% img https://ws2.sinaimg.cn/large/006tKfTcgy1frgdgxjqxuj31kw11x1fa.jpg %}

先至塔尔寺，拜千佛殿，佛像庄严肃穆，双手合十，烛光浮动，梵音不绝于耳，殿外梅花盛开，落英缤纷。离寺，往青海湖，路上雪山连绵不绝，风雪交加，车内歌声嘹亮。同行者皆不惧寒冷，雪地摄影，不亦乐乎。等到青海湖，天空，一半乌云密布，一半白云悠悠，湖水清澈，波光粼粼，海天一线，简直美哉。及黑马河夜宿，冷，翌日清晨无日出，憾之。

午时到茶卡盐湖，入门观之，以为不过如此，待行之三、四公里，咋见湖面如镜，湖中人影清晰可见，远方苍茫一片白，天与地似乎傻傻分不清。同行姑娘入湖，余为之摄影，镜头之内偏偏起舞，绿裙如茵，娇艳似花，与精灵无二，心有感叹：天空之镜果如是。再行驶至翡翠湖，湖水似染，茵茵动人，同游四人让师傅拍摄短片做纪念。离翡翠湖及大柴旦夜宿，一路上亦是风景无数。

{% img https://ws1.sinaimg.cn/large/006tKfTcgy1frgdjvjt95j31kw11xn9c.jpg %}

又清晨至雅丹，山丘无数，错踪而立，苍穹蓝幕，登丘而摄，风凛凛，天之高而人之渺。往阿塞克，一路群山似褐色苍龙，再往阳关，策马古道，绝尘千里，映落日余晖，不时风沙扑来，兴致正高，羌笛声声，旷世苍凉，当真是，西出阳关无故人。不舍离阳关，勉强入鸣沙。到鸣沙村，放行李于月笼沙客栈，至傍晚入沙漠露宿，沙漠飞车，脱鞋滑沙，甚是惊险刺激。夕阳下，围炉而坐，饮酒而歌，篝火烟花，难忘今宵。

{% img https://ws2.sinaimg.cn/large/006tKfTcgy1frgdlt7aguj31kw11xjzh.jpg %}

到天亮，有流星划落天际，不知有人许愿，唤同游者登沙山之顶观日出。及正午，去莫高窟，观古人之巧夺天工。不由感叹，万物皆无永恒，同伴举杯劝慰，永恒者乃思想精神，思之精妙，想之神奇，方能流芳百世，如丝绸之路留给后世之不惧艰险路远，心存信念的精神。傍晚，入鸣沙山，骑骆驼，短短四公里却能体会到古人丝路之不易。

第五日，过瓜州品瓜赏干果，到甘肃丹霞山游玩，余为黔东南云贵高原之人，山已看尽，遂觉得平淡无奇。

最后一日，即将返程，师傅相邀，故作此游记。茫茫人海，五人相聚，实属缘分，青甘环线，不虚此行。（二零一八年五月十三）
